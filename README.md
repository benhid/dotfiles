Modular dotfiles for my personal setup.

## Installation

* Download dotfiles:
   ```sh
    git clone git@github.com:benhid/dotfiles.git
   ```
* Make the script executable:
   ```sh
   chmod +x ./install.sh
   ```
* Run the script:
   ```sh
   ./install.sh
   ```
